Initialize project setting:

* In this project, these three packages need to use the specific version

PM> Install-Package Microsoft.AspNet.Identity.Core -Version 1.0.0

PM> Install-Package Microsoft.AspNet.Identity.EntityFramework -Version 1.0.0

PM> Install-Package Microsoft.AspNet.Identity.Owin -Version 1.0.0

 
* Create a database in the sql server
create a database, name as "fscheduler"
in this database persmission setting, add the dbo as dbowner
* In the project:
in the file "Web.config"
 <connectionStrings>
    <add name="DefaultConnection" connectionString="Data Source=.;Initial Catalog=fscheduler;Integrated Security=SSPI;" providerName="System.Data.SqlClient" />
  </connectionStrings>
  <entityFramework>
    <defaultConnectionFactory type="System.Data.Entity.Infrastructure.LocalDbConnectionFactory, EntityFramework">
      <parameters>
        <parameter value="v11.0" />
      </parameters>
    </defaultConnectionFactory>
  </entityFramework>
 
* in the package manager console
 Enable-Migrations –EnableAutomaticMigrations 
Add-Migration
Update-Database –Verbose
* default administrator account
in the folder of "Migrations", update those data
  // Change these to your own:
